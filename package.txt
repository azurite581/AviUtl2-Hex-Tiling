[ 概要 ]

Practical Real-Time Hex-Tiling (https://jcgt.org/published/0011/03/05/) の実装を AviUtl2 に移植したものです。
六角形でタイリングすることでテクスチャの繰り返しや継ぎ目が目立ちにくくなります。

[ インストールされるファイル ]

・Hex-Tiling.anm2

・English.Hex-Tiling.aul2
翻訳ファイル（英語）

[ 使い方 ]

GitHub リポジトリ(https://github.com/azurite581/AviUtl2-Hex-Tiling) を参照してください。
